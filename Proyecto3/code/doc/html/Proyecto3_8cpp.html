◆ callLiebmann()

void callLiebmann ( config &  configuration)

Call the Liebmann method with the configuration given in the terminal.

Call the Liebmann method to obtain the heat distribution in the plaque.
If the flag to measure the execution time is on, the Liebmann algorithm will be executed but no results will be shown, only prints the execution time in terminal.
If the flag is off, the the algorithm will be executed and if the flag to show results is on, the plot of the result will be shown in a new window.

Parameters
[in]configurationStruct with the parameters entered in terminal
